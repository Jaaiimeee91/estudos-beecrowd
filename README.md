# 🐝 Estudos Beecrowd - Java

Este repositório contém as minhas resoluções dos desafios da plataforma [Beecrowd](https://www.beecrowd.com.br/). O foco principal é praticar a lógica de programação e a sintaxe da linguagem Java.

## 🚀 Tecnologias Utilizadas
* **Java 19**
* **VS Code**
* **Git & GitHub**

## 📂 Estrutura do Repositório
* `/Iniciante`: Problemas da categoria iniciante, focados em sequência lógica, estruturas condicionais e de repetição.

## 🛠️ Como rodar os exercícios
1. Clone o repositório.
2. Certifique-se de ter o JDK instalado.
3. No VS Code, abra o arquivo desejado e clique em **Run**.
